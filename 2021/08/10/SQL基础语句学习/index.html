重要的SQL命令:

  • SELECT - 从数据库提取数据
  • UPDATE - 更新数据库数据
  • DELETE - 从数据库删除数据
  • INSERT INTO - 向数据库插入数据
  • CREATE DATABASE - 创建新数据库
  • ALTER DATABASE - 修改数据库
  • CREATE TABLE - 创建新表
  • ALTER TABLE - 修改数据库表
  • DROP TABLE - 删除表
  • CREATE INDEX - 创建索引
  • DROP INDEX - 删除索引

SELECT DISTINCT

distinct(独特的)

语法:

在表中, 一列可能包含多个重复的值, 该语句即用于返回唯一不同的值.

若只指定一列, 则返回该列中重复的值中的一个.

若指定多列, 则只有多列都完全相同的项才不会输出, 若在列1相同的项, 在列2不同, 则这两项都会输出.

WHERE子句中的运算符:

运算符 描述
= 等于
<> 不等于。注释：在 SQL 的一些版本中，该操作符可被写成 !=

LIKE 搜索某种模式

WHERE 子句 AND & OR

  • AND 即条件运算的’与’
  • OR 即条件运算的’或’

WHERE子句的条件表达式, 可以使用圆括号显式标明优先.

也存在NOT, 即条件运算的’非’, 默认NOT优先级最高, 其次为AND, 再为OR
